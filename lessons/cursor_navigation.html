Text cursor navigation

Placing characters in a text box is great, but you most likely want to move the cursor around before placing them. Normally you're used to moving the cursor around with the arrow keys and with the mouse. In this lesson your goal is to correct the mistakes by moving the cursor around and placing the correct letters you have learned from the alphabet.
If you wish to leave this lesson, you can press the 'Close' button, or simply say Control Whale to press Ctrl+W to close this tab.

If you have selected this lesson using voice commands, you've already used one or more of the commands that move the cursor around. go left and go right are both used to press an arrow key either to the left or the right. But you also have go up and go down to move vertically.
The word go has plenty of other navigation options. For example, saying go line start and go line end will move the cursor to the start or the end of the line.
Skipping to the start or end of the file is as easy as saying go way up and go way down
Saying go word left moves the cursor to the start of the current word, or moves it over to the start of the previous word, while go word right moves the cursor to the end.

These navigation commands can be found here: core/edit/edit.talon

Naturally, you can keep repeating these commands until you get to your desired location, but you can also let talon repeat them for you. Adding an ordinal number behind a command repeats it. So saying go right seventh will move the cursor right seven times. All the ordinal code can be found here: core/numbers/ordinals.py
