Step one of copying and pasting is the selection. Talon offers a whole range of different ways of selecting characters, lines and words. Like moving your arrow keys around using the phrases starting with go, and deleting things using the word clear, we can start a selection using the word select.
select all for example will select everything in a text box. There's a ton of variations to this, like select line to select your current line, and select word to select your current word.
If you remember the previous lesson, where we looked at clearing, we can also add a direction, like select word left to select the word to the left of the cursor. Or even select up to move your cursor up while selecting things.
If you need more precise controls, like selecting a special character, you can use select right and select left to select single characters. You keep saying select phrases until you are satisfied with your current selection. You can find all these commands here: core/edit/edit.talon.

Step two and three, the copying or cutting, and pasting of your selection, is are pretty self explanatory. The only commands you need to remember are copy that, cut that and paste that. These commands can be found here: core/edit/edit.talon

Once you are comfortable with these controls, you can start getting fancy with combining step one and two. For example, copy word left will select the word to the left, and then copy it straight away.
